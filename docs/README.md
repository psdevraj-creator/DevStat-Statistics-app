# DevStat — Statistics App

[![User Guide](https://img.shields.io/badge/%F0%9F%93%96-User_Guide_%26_Statistics_Manual-005eb8?style=for-the-badge)](https://psdevraj-creator.github.io/DevStat-Statistics-app/USER_GUIDE_AND_STATISTICS_MANUAL.html)

A desktop medical statistics application with a point-and-click interface. Analyse data, run statistical tests, and create publication-quality charts — all locally on your machine.

## Features

- Upload CSV, Excel, and SPSS files
- Descriptive statistics, t-tests, ANOVA, correlation, regression
- Survival analysis (Kaplan-Meier, Cox regression)
- Diagnostic test evaluation (sensitivity, specificity, ROC curves)
- Factor analysis and reliability (Cronbach's alpha)
- 37+ interactive chart types
- Eligibility engine to guide analysis choices

## Quick Start

1. Download the latest release from [Releases](https://github.com/psdevraj-creator/DevStat-Statistics-app/releases)
2. Unzip and run `DevStat.exe`
3. Open `http://localhost:8150` in your browser
4. Upload your data and start analysing
